주택청약 정보알리미에 들어가면 청약공고, 청약 유형별 정보 탭이 뜹니다.

청약공고 블록에 공고(수도권) 버튼을 누르면 지역을 입력해주세요라는 질문이 생성됩니다.

이용자가 원하는 지역을 입력하였을때 공고가 있을 경우 정보를 제공하고 
그에 맞는 정보를 제공하고 없을시에는 현재 모집중인 공고가 없습니다 라고 뜹니다.
만약 다른 공고들을 더 보고 싶을 시에는 하단에 다른 공고 더보기 버튼을 클릭하여 
청약home 홈페이지로 이동하여 정보를 얻습니다.
잘못 입력했을 시 챗봇에서 무엇을 원하는지 모르겠어요, 이해하기 어려워요 등의 질문이 생성됩니다. 
정보가 나오지 않을 시 다시 지역을 입력하여 정보를 얻습니다.

공고부분이 모두 끝나고 청약 유형별 정보에 대해 궁금하면 시작하기(처음으로) 버튼을 클릭합니다.

청약 유형별 정보에는 공공분양, 민간분양 버튼이 있습니다.

공공분양 버튼을 클릭 시 일반 분양, 특별분양, 우선분양 세가지 버튼을 가진 블록이 생성됩니다.
일반 분양 클릭 시 간단한 정보가 있는 카드가 나타나고, 하단에 순위요건 버튼이 생성됩니다.
클릭 시 공공분양 1순위 요건, 동일 순위 경쟁 탭이 생성됩니다. 
정보가 간단하다고 생각될 시 자세히 보기버튼을 클릭하여 주택공급에 관한 법령으로 이동하여 정보를 습득합니다.
공공분양 유형 특별분양을 클릭하면 신혼부부,생애최초, 다자녀가구, 노부모 부양, 기관추천 총 다섯개의 블록이 간단한 정보와 함께 생성됩니다. 
간단한 정보가 제공되었으나, 더 궁금한 점이 있을 시 하단에 자세히 보기 버튼을 클릭합니다
버튼 클릭을 하게되면 찾기 쉬운 법령정보로 이동하여 정보를 얻습니다.
공공분양 유형 우선분양을 클릭하면 직장 주택 조합, 주상복합 건축물의 건설부지 소유자, 
대규모 택지 개발지구 총 세개의 블록이 간단한 정보와 함께 생성됩니다.
사용방법은 위와 동일합니다.

민간분양 유형을 클릭하면 공공분양과 동일하게 총 세개의 블록이 생성됩니다.
일반공급 클릭 시 간단한 조건과 하단에 순위 요건 버튼이 생성됩니다.
클릭 시 1순위 요건, 통장예치금액표가 생성되고, 하단에 계산기 사용하러가기 버튼이 생성됩니다.
클릭 시 가점제 점수계산표 버튼이 이미지와 같이 생성됩니다. 
클릭 시 조건 1 블록이 가점표와 함께 생성됩니다. 하단의 계속하기 버튼을 클릭 시 
조건 1에 해당하는 값을 입력해주세요라는 질문이 생성됩니다.
위의 표에 나오는 자격에 맞는 입력값을 입력하면 조건 2로 넘어갑니다.
위와 동일하게 수행하고, 만약 잘못 입력하였을 시 이전으로 버튼을 클릭하여 수정합니다.
위와 동일하게 조건 3까지 수행하면 가점제/ 추천제 표와 함께 총합 점수가 계산됩니다.

특별분양 시 일반분양과 동일하게 5가지의 유형의 블록이 간단한 정보와 함께 생성됩니다. 
더 많은 정보를 보기를 원한다면 하단의 자세히보기 버튼을 클릭합니다.
버튼 클릭을 하게되면 찾기 쉬운 생활 법령 정보로 이동하여 정보를 얻습니다.

위와 동일하게 더 많은 정보를 보기 원한다면 하단의 자세히보기 버튼을 클릭합니다.
